Глоссарий по линейному программированию

Для удобства навигации глоссарий разделен на разделы, соответствующие главам курса.

Глава 1. Основные понятия

Базисное допустимое решение (БДР)
Базисное решение задачи линейного программирования, которое также является допустимым (удовлетворяет ограничениям по неотрицательности переменных).
Базисное решение задачи ЛП
Решение задачи ЛП, в котором векторы-столбцы матрицы ограничений, соответствующие всем ненулевым координатам этого решения, образуют линейно независимую систему векторов.
Выпуклая комбинация точек
Точка $x = \alpha x^1 + (1 - \alpha) x^2, 0 \leq \alpha \leq 1$, где $x^1$ и $x^2$ - две точки в $R^n$. Представляет собой отрезок, соединяющий точки $x^1$ и $x^2.
Выпуклая функция
Функция $f(x)$ на выпуклом множестве $X$, удовлетворяющая условию: $f(\alpha x' + (1 - \alpha) x'') \leq \alpha f(x') + (1 - \alpha) f(x''), 0 \leq \alpha \leq 1$ для всех $x', x'' \in X$.
Выпуклое множество
Множество $X \subseteq R^n$, которое вместе с любыми двумя своими точками содержит и отрезок, соединяющий эти точки.
Выпуклое многогранное множество
Множество, являющееся конечным пересечением гиперплоскостей и полупространств. Также может быть представлено как множество выпуклых комбинаций конечного числа точек и конуса.
Вырожденная задача ЛП
Задача линейного программирования, имеющая хотя бы одно вырожденное допустимое базисное решение.
Вырожденное допустимое базисное решение
Допустимое базисное решение, в котором число ненулевых координат строго меньше, чем $m=rk(A)$.
Допустимое решение задачи ЛП
Решение задачи линейного программирования, удовлетворяющее ограничениям по неотрицательности переменных ($x \geq 0$).
Задача линейного программирования (ЗЛП)
Задача оптимизации, в которой целевая функция и ограничения являются линейными.
Каноническая форма задачи ЛП
Форма записи задачи линейного программирования, в которой ограничения представлены в виде равенств, все переменные неотрицательны, и требуется минимизировать линейную целевую функцию.
Крайняя точка выпуклого множества
Точка выпуклого множества, которая не является внутренней точкой никакого отрезка, целиком принадлежащего этому множеству.
Линейное программирование (ЛП)
Раздел математического программирования, изучающий методы решения задач оптимизации с линейной целевой функцией и линейными ограничениями.
Матрица питательности
Матрица $A=[a_{ij}]$, где $a_{ij}$ - количество $i$-го компонента, содержащегося в единице веса $j$-го продукта (в задаче о диете).
Матрица урожайности
Матрица $A=[a_{ij}]$, где $a_{ij}$ - урожайность $j$ культуры на $i$ поле (в задаче о посевной площади).
Оптимальное решение задачи ЛП
Допустимое решение задачи линейного программирования, доставляющее минимум (или максимум) целевой функции.
Область допустимых решений (ОДР)
Множество всех допустимых решений задачи линейного программирования. Геометрически представляет собой выпуклое многогранное множество.
Решение задачи ЛП
Вектор $x$, удовлетворяющий системе линейных уравнений ограничений задачи ЛП.
Стандартная форма задачи ЛП
Форма записи задачи линейного программирования, в которой ограничения представлены в виде неравенств типа "$\geq$" (или "$\leq$"), все переменные неотрицательны, и требуется минимизировать (или максимизировать) линейную целевую функцию.
Технологическая матрица
Матрица $A=[a_{ij}]$, где $a_{ij}$ - количество $i$-го ресурса, необходимое для производства единицы продукции $j$ (в общей производственной задаче).
Целевая функция
Функция $f(x_1, x_2, ..., x_n)$, которую требуется оптимизировать (минимизировать или максимизировать) в задаче линейного программирования.
Задача ЛП в общей форме
Задача линейного программирования, в которой ограничения могут быть представлены как в виде неравенств, так и в виде равенств, и на часть переменных могут быть не наложены условия неотрицательности.

Глава 2. Симплекс-метод

Ведущая строка
Строка симплекс-таблицы, соответствующая переменной, выводимой из базиса на текущей итерации симплекс-метода.
Ведущий столбец
Столбец симплекс-таблицы, соответствующий переменной, вводимой в базис на текущей итерации симплекс-метода. Выбирается на основе оценок $\Delta_j$.
Ведущий элемент
Элемент симплекс-таблицы, находящийся на пересечении ведущей строки и ведущего столбца. Используется для пересчета симплекс-таблицы.
Искусственный базис
Векторы, добавляемые к исходной системе ограничений для получения начального базисного допустимого решения при использовании метода искусственного базиса.
Искусственные переменные
Переменные, добавляемые к исходной системе ограничений для получения начального базисного допустимого решения в методе искусственного базиса. В оптимальном решении должны быть равны нулю.
М-метод искусственного базиса
Вариант метода искусственного базиса, в котором искусственные переменные вводятся в целевую функцию с большим штрафным коэффициентом $M$, чтобы алгоритм стремился исключить их из базиса.
Оценка $\Delta_j$
Величина, характеризующая эффективность введения небазисной переменной $x_j$ в базис на текущей итерации симплекс-метода. Рассчитывается как $\Delta_j = z_j - c_j = (c_б, A_j) - c_j$.
Симплекс-метод
Итерационный алгоритм решения задач линейного программирования, основанный на направленном переборе базисных допустимых решений до нахождения оптимального решения или установления неограниченности целевой функции.
Симплекс-таблица
Табличная форма представления данных задачи линейного программирования на каждой итерации симплекс-метода, используемая для удобства расчетов и выбора переменных для ввода и вывода из базиса.
Холостой ход симплекс-метода
Ситуация в симплекс-методе при решении вырожденной задачи, когда значение целевой функции не меняется, но происходит замена базиса. Может привести к зацикливанию.

Глава 3. Двойственность в линейном программировании

Двойственная задача
Задача линейного программирования, построенная определенным образом на основе исходной (прямой) задачи. Пары двойственных задач обладают важными свойствами, связывающими их решения.
Первая теорема двойственности
Теорема, утверждающая, что для пары двойственных задач выполняется одна из трех альтернатив: обе задачи имеют оптимальные решения с равными значениями целевых функций; одна задача имеет неограниченную целевую функцию, а другая не имеет допустимых решений; обе задачи не имеют допустимых решений.
Принципы двойственности
Набор правил, определяющих, как построить двойственную задачу по заданной прямой задаче линейного программирования. Включают правила транспонирования матрицы ограничений, изменения знаков неравенств и перестановки коэффициентов целевой функции и правых частей ограничений.
Условия дополняющей нежесткости
Необходимые и достаточные условия оптимальности допустимых решений прямой и двойственной задач. В слабой форме утверждают, что на оптимальных решениях скалярное произведение векторов дополнительных переменных и векторов невязок ограничений равно нулю.
Теневые цены
Оптимальные значения переменных двойственной задачи, интерпретируемые как оценки "ценности" ресурсов в задаче линейного программирования. Показывают, на сколько изменится оптимальное значение целевой функции при изменении запаса соответствующего ресурса на единицу.
